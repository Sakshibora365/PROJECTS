# 🌿 GlowSkin - Natural Skincare Landing Page

A clean, responsive landing page built for a fictional skincare brand **GlowSkin** using HTML, CSS, and Bootstrap 5.

---

## 🔗 Live Demo
https://glowskincareproject.netlify.app/ 


## ✨ Features
- Fully responsive design (mobile + tablet + desktop)
- Sticky navigation bar
- Hero section with CTA button
- Features section with hover effects on cards
- Contact form
- Footer with social media icons
- Smooth scrolling

---

## 🛠️ Built With
- HTML5 (Semantic structure)
- CSS3 (Custom styling + animations)
- Bootstrap 5 (Grid, components, icons)
- Google Fonts (Poppins)

---

## 📁 Project Structure
basic-landing-page/
├── index.html       → Main HTML file

## 🚀 How to Run
1. Clone the repo
```bash
git clone https://github.com/yourusername/glowskin-landing-page
```
2. Open `index.html` in your browser
3. That's it! No installation needed ✅
